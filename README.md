## 📒 Descrição
Este e-book de 3 páginas apresenta o impacto da Organização doméstica na vida das pessoas. 

## 🤖 Tecnologias Utilizadas
* IA Generativa ChatGPT para roteirização
* Aplicativo CANVA para criar a arte visual do e-book
* Ferramenta Power Point para edição

## 🧐 Processo de Criação
Utilizei o ChatGPT para revisar a minha pesquisa sobre a Organização doméstica e rotinas. Utilizei também o CANVA para gerar as imagens ilustrativas do e-book e também a capa. Por fim, a montagem foi feita no Power Point.

## 🚀 Resultados
O e-book apresenta um conteúdo inovador para pessoas comuns de qualquer área de conhecimento podendo ser utilizado como uma ferramenta importante de organização de vida na  área de pessoal e profissional além de ajudar na realização de novos projetos, sempre apresentado como fosso o usuário final.
